# Cupcake Confections LLC – Prototype & Usability Testing

## Overview

Designed an e-commerce cupcake ordering prototype focused on improving user navigation, checkout efficiency, and customer experience.

## Objectives

* Create an intuitive cupcake ordering workflow
* Improve checkout usability
* Design clear user navigation flows
* Conduct usability testing and gather feedback

## Features

* Window Navigation Diagram (WND)
* Sequence Diagram
* Product selection workflow
* Shopping cart and checkout process
* Payment interface prototype
* Order confirmation flow

## Tools Used

* Visio
* Microsoft Office
* UI/UX Design Concepts
* Process Modeling

## Usability Testing

Conducted usability testing with multiple users to evaluate:

* ease of navigation
* checkout experience
* button clarity
* interface organization

## Outcome

Users reported that the application was easy to use, visually appealing, and well organized.
